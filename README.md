C++ implementation of a class that represents real numbers with any amount of digits without precision loss.
Status: In development.

Current and future features:
	+ Addition/Subtraction of real numbers
	+ Multiplication of real numbers
	- Division of real numbers
	- Some math functions (sin, cos, tan, log, e^x, a^x, sqrt, ...)
	- Different rounding methods

Notes:
	Features with a minus at the beginning of the line aren't implemented yet.
	Features with a plus at the beginning of the line are implemented
